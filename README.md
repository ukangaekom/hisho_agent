# Hisho Agent

















## Installation and Running Locally

```
    git clone 
```

### Setting up of Enviroments
```
    TELEGRAM_BOT_TOKEN=<TOKEN_FOR_TELEGRAM_BOT>
    GEMINI_API_KEY=<API_KEY_FOR_GEMINI>
    MODEL_GEMINI2="gemini-2.0-flash";

```

Run Cargo 

```
    cargo build; cargo run;
```
